# Anne's Dieren Tekeningen

Een professionele portfolio website voor Anne's Dieren Tekeningen.

## Over het project

Deze website showcases handgemaakte tekeningen van dieren, gemaakt door Anne. De site heeft een warme, vriendelijke uitstraling met zachte pastelkleuren en is volledig responsive.

## Functionaliteiten

- **Responsive design**: Werkt perfect op desktop, tablet en mobiel
- **Galerie met lightbox**: Klik op afbeeldingen om ze groter te bekijken
- **Smooth scrolling**: Vloeiende navigatie tussen secties
- **Moderne uitstraling**: Zachte schaduwen, ronde hoeken en warme kleuren

## Technologie

- Pure HTML5
- CSS3 (geen frameworks)
- Vanilla JavaScript
- Google Fonts (Poppins)

## Bestandsstructuur

```
.
├── index.html          # Hoofdpagina
├── style.css           # Styling
├── script.js           # JavaScript functionaliteit
├── images/             # Galerie afbeeldingen
└── README.md           # Deze file
```

## Lokaal draaien

Open `index.html` in je browser, of gebruik een lokale webserver:

```bash
python -m http.server 5000
```

Ga vervolgens naar `http://localhost:5000`

## GitHub Pages deployment

1. Push de code naar een GitHub repository
2. Ga naar Settings > Pages in je repository
3. Selecteer de main branch als bron
4. Je website is nu live op `https://gebruikersnaam.github.io/repository-naam`

## Aanpassingen

### Afbeeldingen vervangen

- Vervang de bestanden in de `images/` map met eigen tekeningen
- Houd dezelfde bestandsnamen aan (placeholder-1.jpg t/m placeholder-6.jpg)
- Of pas de bestandsnamen aan in `index.html`

### Facebook link toevoegen

Zoek in `index.html` naar `href="#"` bij de Facebook knoppen en vervang `#` door je Facebook pagina URL.

### E-mailadres aanpassen

Zoek in `index.html` naar `mailto:info@example.com` en vervang dit met het juiste e-mailadres.
